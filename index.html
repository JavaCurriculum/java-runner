Java 実行練習

コードを書き換えて、実行ボタンを押してください。 複数クラス・パッケージ構成でも実行できます。

確認ポイント

  • 最初は src/Main.java だけです
  • クラスを追加したい場合は ＋ クラス追加 を押します
  • パッケージを作りたい場合は ＋ パッケージ を押します
  • 例：com.example を作ると src/com/example として表示されます
  • パッケージ内のクラスを使う場合は、Main.java 側で import してください

使い方

  1. src/Main.java に実行開始の処理を書く
  2. 必要に応じてパッケージやクラスを追加する
  3. 左のプロジェクト構成から編集したいファイルを選ぶ
  4. 実行 を押す
