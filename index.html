Java 実行練習

コードを書き換えて、実行ボタンを押してください。 複数クラス・パッケージ構成でも実行できます。

確認ポイント
  • 最初は src/Main.java だけです
  • ＋ パッケージ で com.example などを作れます
  • ＋ クラス追加 でJavaファイルを追加できます
  • ファイルはフォルダへドラッグ＆ドロップで移動できます
  • ドラッグが難しい場合は、ファイルを選んで 場所移動 を押してください
  • Main.java も移動できます。移動時は実行クラスも自動で変わります
  • 候補が出ない場合は Ctrl + Space を押してください
使い方
  1. 実行開始のクラスに main メソッドを書く
  2. 必要に応じてパッケージやクラスを追加する
  3. 左の構成ツリーから編集ファイルを選ぶ
  4. 不要なファイルやパッケージは選択して 削除 を押す
  5. 実行 を押す
